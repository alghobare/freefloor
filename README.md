# freefloor Courses Page

This is a dedicated HTML page for offering training courses on the freefloor website.

## Usage

1. Open `courses.html` in a web browser to view the courses page.
2. The page includes sample courses with enrollment buttons (currently showing alerts).
3. Navigation links back to the home page (`index.html`).

## Features

- Responsive design with basic CSS.
- Sample course listings with details.
- Placeholder enrollment functionality using JavaScript alerts.

## Troubleshooting

- If styles don't load, ensure the browser supports inline CSS.
- For production, consider moving CSS to `style/courses.css` and JS to `js/courses.js`.
- No server required; this is a static HTML page.

## Customization

- Edit the course details in the HTML.
- Add more courses by duplicating the `.course` divs.
- Integrate with a backend for real enrollment if needed.
